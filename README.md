InAmigos Foundation - NGO Awareness Website

A responsive NGO Awareness Website developed using HTML5 and CSS3 to spread awareness about the initiatives, projects, and social impact of InAmigos Foundation.

🌍 Project Overview

This website was created as part of an internship task to showcase the mission, vision, ongoing projects, and community impact of InAmigos Foundation.

The website aims to encourage people to participate in social activities by volunteering, supporting campaigns, and contributing to community development.
